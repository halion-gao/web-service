## 🛠️ 技術實現亮點

1. **純 Web Audio API 鋼琴八音盒多聲部合成器** 🎵
   - 專案內無引入任何 mp3 檔案。利用瀏覽器 Web Audio API 三角波（Triangle Wave）動態模擬鋼琴鋼 comb 撥弦音色。
   - 內置溫暖的 Delay（延遲）與 Feedback（回饋）共鳴節奏電路。
   - **多聲部伴奏**：為主旋律增加了溫暖的低音和弦伴奏（如 C, F, G 大三和弦的根音與分解和弦），使樂聲聽起來更飽滿。
   - **動態音效合成**：打字機聲（帶帶通濾波器的白噪聲爆發）、氣泡戳破聲（快速指數下降的 Sine 波掃頻）與星星亮光聲（C大調五度音階延遲琶音）全部採用純程式碼即時合成。

2. **60 FPS 高效 Canvas 櫻花與愛心飄雪引擎** 🌸
   - 使用 HTML5 Canvas 與 `requestAnimationFrame` 繪製高流暢度飄落特效。
   - 包含正弦波風力擺動（Sway）物理計算，模擬真實櫻花瓣與愛心氣球的慢速滑翔漂移效果。

3. **互動式 Canvas Parallax 星空背景** 🌌
   - 透過 Canvas 繪製 80-120 顆帶有 Twinkle 隨機閃爍動畫的星星，且星星會隨著**滑鼠在畫面的移位產生視差偏移**，增添夜空的深邃感。

4. **現代 CSS 獨立 Scale 屬性優化** ⚡
   - 氣泡縮放使用獨立的 `scale` 屬性，避免與 JavaScript 物理引擎更新位置用的 `transform: translate3d()` 發生屬性覆蓋衝突，杜絕任何畫面跳動，並提升 GPU 渲染效能。
   - 使用原生 `<dialog>` 搭配全新的 `@starting-style`，流暢過渡彈出視窗的淡入與放大動畫。

## 🚀 快速開始

### 1. 本地開發 (免安裝 Node 方案)
由於專案已優化為原生相容性，且音效全部由瀏覽器即時合成，如果本機未安裝 Node.js，可以直接透過 Python 開啟本地服務預覽：
```bash
# 開啟 Python 本地伺服器
py -m http.server 5173
```
開啟後在瀏覽器訪問 `http://localhost:5173/` 即可。

### 2. Node.js 開發
如果已安裝 Node.js，可使用 Vite 啟動：
```bash
npm install
npm run dev
```

### 3. 生產環境打包
將專案打包成最佳化靜態檔案：
```bash
npm run build
```
編譯完成後，檔案會輸出至 `dist/` 目錄，可直接部署至 Github Pages、Vercel 或 Firebase Hosting 上。

### 4. Docker 容器化部署

本專案支援使用 Docker 進行輕量化的 Nginx 容器化部署。

#### 本地打包鏡像
在專案根目錄下執行以下指令打包映像檔：
```bash
docker build -t ckc101-web:latest .
```

#### 上傳至 Docker Hub
登入 Docker Hub 並為映像檔加上標籤，然後推送到您的遠端倉庫：
```bash
# 登入 Docker Hub
docker login

# 標記映像檔 (請將 <username> 替換為您的 Docker Hub 帳號，例如 halion0329)
docker tag ckc101-web:latest <username>/lyn-web:latest

# 推送至遠端倉庫
docker push <username>/lyn-web:latest
```

#### 運行容器
在任何安裝了 Docker 的環境中執行以下指令即可啟動服務：
```bash
# 以背景模式運行，並將容器的 80 通訊埠對應至本機的 8080 通訊埠
docker run -d -p 8080:80 --name lyn-birthday-web <username>/lyn-web:latest
```
運行後在瀏覽器造訪 `http://localhost:8080` 即可檢視網頁。
